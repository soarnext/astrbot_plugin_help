# 更新日志 (CHANGELOG)

## [v1.0.4] - 2026-03-29

### 🔧 优化与修复
- **头像路径解析增强**：引入了更健壮的 `get_image_data_uri` 函数，支持 URL、绝对路径、以及相对于基础目录和插件数据目录的多种路径解析方式。
- **配置校验优化**：在插件初始化阶段增加了对 `bot_name` 和 `help_avatar` 的类型校验。
- **多头像随机切换**：如果配置了多个头像路径，插件将随机选择其中一个进行显示，增加菜单多样性。
- **模板读取优化**：使用 `pathlib` 改进了 HTML 模板的读取方式，增加了错误处理机制。

## [v1.0.3] - 2026-03-29

### 🔧 优化与修复
- **路径解析逻辑修复**：修复了在部分环境下自定义头像路径无法正确加载的问题。
- **MIME 类型识别**：优化了图片 MIME 类型识别逻辑，支持更多图片格式。

## [v1.0.2] - 2026-03-29

### ✨ 新特性
- **布局自适应优化**：进一步优化了响应式网格布局，调整了最小宽度限制（800px）和指令卡片大小，确保在指令较多时能自动切换为多列展示，彻底解决单列排版和右侧留白问题。
- **权限管理增强**：实现“自动识别+手动覆盖”逻辑。插件现在能自动识别内置指令的 `(admin)` 标记和其它插件的管理员权限过滤器，同时支持通过 `admin_commands` 和 `user_commands` 强制调整。
- **标题自动化**：合并标题配置为 `bot_name`，自动生成“<名称>帮助文档”或“<名称>管理帮助文档”。
- **内置指令优化**：内置指令默认开启，且无需手动设置即可自动根据权限分发到对应的菜单。
- **排版优化**：管理员帮助菜单的副标题现在支持留空，留空后自动隐藏并采用紧凑排版。

### 🔧 优化与修复
- **头像读取修复**：优化了头像路径解析逻辑，现在支持绝对路径、插件相对路径和运行目录相对路径，并自动识别图片格式（PNG/JPG/WEBP），彻底解决自定义头像不生效的问题。
- **重命名**：插件正式更名为 `astrbot_plugin_xirohelp`。
- **语法修复**：修复了 `main.py` 中异步生成器使用 `yield from` 的语法错误。
- **兼容性修复**：修复了 `PermissionFilter` 在某些版本下导入失败的问题，改用更兼容的属性检测机制。
- **代码清理**：移除了过时的 `draw.py` 绘图逻辑及相关字体文件，全面转向 HTML 渲染。

## [v1.0.1]

### ✨ 新特性
- **功能分离**：新增 `/adminhelps` 指令，实现普通用户菜单与管理员菜单的严格分离。
- **头像支持**：支持自定义帮助菜单左上角的头像，支持本地上传或网络 URL。
- **名称映射**：支持通过配置将插件原始名称映射为自定义展示名称。

## [v1.0.0]

### 🎉 初始版本
- 基于 HTML 渲染的高颜值帮助菜单插件。
- 自动扫描已激活插件并生成指令列表。
- 支持插件黑名单及自定义额外指令。
